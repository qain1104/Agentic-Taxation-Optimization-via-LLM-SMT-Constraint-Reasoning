business_income_tax.py ─ 營利事業所得稅（DEFAULTS / 級距與稅率）
================================================================================

- input 參數預設值：所有收支/扣除欄位預設為 0
  - OperatingRevenueTotal, SalesReturn, SalesAllowance, OperatingCost,
    OperatingExpensesLosses, NonOperatingRevenueTotal, NonOperatingLossExpenses,
    Prev10LossDeduction, TaxIncentiveExempt, ExemptSecuritiesIncome,
    ExemptLandIncome, Article4_4HouseLandGain

- 預設期間參數
  - is_full_year: True
  - m_partial: 12  （僅在 is_full_year=False 時使用，代表課稅月份數）

- 稅負計算級距常數（程式碼內變數 FULL1 / FULL2）
  - FULL1: 120,000
  - FULL2: 200,000

- 稅額公式（以課稅所得額 P 為基礎；is_full_year=True）
  - P ≤ 120,000 → tax = 0
  - 120,000 < P ≤ 200,000 → tax = (P - 120,000) × 0.5  （即半數課稅）
  - P > 200,000 → tax = P × 0.2  (20%)

- 非全年（is_full_year=False）調整方式
  - P_adj = P × 12 / m_partial
  - 先以 P_adj 套用上述級距求稅，再乘回 (m_partial / 12)

- P 計算中「直接扣除」的項目（以 input 參數名稱表示）
  - Prev10LossDeduction（前 10 年虧損扣除）
  - TaxIncentiveExempt（租稅優惠免稅/減免）
  - ExemptSecuritiesIncome（證券交易免稅所得）
  - ExemptLandIncome（土地交易免稅所得）
  - Article4_4HouseLandGain（所得稅法第 4 條之 4 房地合一相關）
